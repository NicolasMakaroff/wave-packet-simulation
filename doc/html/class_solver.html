Detailed Description

Solver for a time-dependent non-relatistic Schrödinger equation using FTCS, BTCS and CTCS scheme.

Parameters
container_the type of the matrix of which we are computing the COD.
potentials_
shift_operator_This class performs a PDE resolution using one of the three scheme implemented : FTCS, Forward-Time Central-Space & BCTS, Backward Time, Centered Space & CTCS Center-Time Central-Space. The formula are recall for each scheme :

FTCS scheme :

\[ \frac{i\hbar}{dt}\Psi^{(t+dt)} = \frac{-\hbar^2}{2m.dx^2}(\Psi_{(x+dx)} + \Psi_{(x-dx)}) - \frac{\hbar^2}{2m.dy^2}(\Psi_{(y+dy)} + \Psi_{(y-dy)}) + (V(x,y)+\frac{i\hbar}{dt} + \frac{\hbar^2}{m.dx^2} + \frac{\hbar^2}{m.dy^2})\Psi \]

BCTS scheme :

\[ (\frac{i\hbar}{dt} - V(x,y) - \frac{\hbar^2}{m.dx^2} - \frac{\hbar^2}{m.dy^2})\Psi^{(t+dt)} = \frac{-\hbar^2}{2m.dx^2}(\Psi_{(x+dx)}^{(t+dt)} + \Psi_{(x-dx)}^{(t+dt)}) - \frac{\hbar^2}{2m.dy^2}(\Psi_{(y+dy)}^{(t+dt)} + \Psi_{(y-dy)}^{(t+dt)}) + \frac{i\hbar}{dt}\Psi \]

CTCS scheme :

\[ (\frac{2i\hbar}{dt} - V(x,y) - \frac{\hbar^2}{m.dx^2} - \frac{\hbar^2}{m.dy^2})\Psi^{(t+dt)} \frac{-\hbar^2}{2m.dx^2}(\Psi_{(x+dx)} + \Psi_{(x-dx)}) - \frac{\hbar^2}{2m.dy^2}(\Psi_{(y+dy)} + \Psi_{(y-dy)}) \frac{-\hbar^2}{2m.dx^2}(\Psi_{(x+dx)}^{(t+dt)} + \Psi_{(x-dx)}^{(t+dt)}) - \frac{\hbar^2}{2m.dy^2}(\Psi_{(y+dy)}^{(t+dt)} + \Psi_{(y-dy)}^{(t+dt)}) + (V(x,y)+\frac{2i\hbar}{dt} + \frac{\hbar^2}{m.dx^2} + \frac{\hbar^2}{m.dy^2})\Psi \]
